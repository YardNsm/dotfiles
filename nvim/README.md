# Neovim configurations

```
nvim.conf
├── after            Loaded after the files in `./plugin`.
│   ├── ftplugin     Configurations for sepcific filetypes.
│   └── syntax       Syntax files.
├── autoload         Autoload runtime files.
├── ftdetect         Setting the filetype for files.
├── plugged          `vim-plug`'s folder.
└── plugin           Plugin-related configurations.
```
